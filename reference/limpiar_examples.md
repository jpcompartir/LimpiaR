# Tibble of examples for LimpiaR functions

Tibble of examples for LimpiaR functions

## Usage

``` r
data("limpiar_examples")
```

## Format

A tibble with 10 rows and 5 columns

- doc_id:

  Post ID

- author_name:

  The author's public username

- mention_content:

  Text variable

- mention_url:

  Link to original post

- platform_interactions:

  Number of interactions on parent platform
